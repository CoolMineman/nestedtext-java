# nestedtext-java
Simple No Dependency NestedText Library For Java 8+

# Features
* Passes all reading tests from [nestedtext_tests](https://github.com/kenkundert/nestedtext_tests) not expected to fail (tests with load_out.err not used)
* No dependencies
* Simple API
